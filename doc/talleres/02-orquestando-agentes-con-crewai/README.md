# Taller 2. Orquestando agentes con CrewAI

## Objetivo
Coordinar varios agentes mediante tareas y flujos para representar un proceso básico de trabajo.

## Contenido sugerido
- definición de roles
- creación de tasks
- configuración de crews
- secuencia de ejecución y trazabilidad

## Relación con el proyecto
Apoya el diseño del flujo de trabajo de una startup de desarrollo de software asistida por agentes.
