# Taller 5. Validación del MVP empresarial

## Objetivo
Aplicar la combinación tecnológica seleccionada sobre un caso de MVP empresarial y documentar resultados técnicos.

## Contenido sugerido
- definición del caso piloto
- implementación base
- pruebas y validación
- registro de hallazgos y limitaciones

## Relación con el proyecto
Soporta la validación inicial del modelo operativo y de la selección tecnológica.
