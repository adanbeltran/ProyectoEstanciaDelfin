# Taller 3. Selección contextual de frameworks

## Objetivo
Estructurar la comparación y selección de framework de agentes y framework de desarrollo para el contexto del proyecto.

## Contenido sugerido
- criterios de selección
- rúbrica o matriz comparativa
- evaluación de alternativas
- justificación de la combinación elegida

## Relación con el proyecto
Responde al objetivo de selección tecnológica de la investigación.
