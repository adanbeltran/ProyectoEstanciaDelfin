# Taller 1. Conociendo agentes individuales CrewAI

## Objetivo
Comprender la creación y ejecución de agentes individuales en CrewAI mediante pruebas separadas por script.

## Contenido sugerido
- instalación del entorno
- configuración de credenciales
- primer agente simple
- agentes independientes para análisis, HTML, CSS, JavaScript y QA

## Relación con el proyecto
Sirve como base técnica para introducir el uso de agentes de IA antes de pasar a orquestación y validación.
